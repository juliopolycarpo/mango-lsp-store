# svelte-lsp

Svelte language server for Svelte components.

## Installation

### Install `svelte-language-server` globally

- With npm:

```bash
npm install -g @svelte/language-server typescript
```

- With bun (Recommended):

```bash
bun install -g @svelte/language-server typescript
```

### Or just add your `svelte-language-server` binary to `PATH` if you have it installed locally.

## Command

```text
svelteserver --stdio
```

## Supported Extensions

`.svelte`

## Runtime Requirement

Install `svelte-language-server` and `typescript` so `svelteserver` is available on `PATH`.
