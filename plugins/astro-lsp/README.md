# astro-lsp

Astro language server for Astro components.

## Installation

### Install `@astrojs/language-server` globally

- With npm:

```bash
npm install -g @astrojs/language-server typescript
```

- With bun (Recommended):

```bash
bun install -g @astrojs/language-server typescript
```

### Or just add your `astro-ls` binary to `PATH` if you have it installed locally.

## Command

```text
astro-ls --stdio
```

## Supported Extensions

`.astro`

## Runtime Requirement

Install `@astrojs/language-server` and `typescript` so `astro-ls` is available on `PATH`.
