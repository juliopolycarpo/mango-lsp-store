# vue-lsp

Vue language server for Vue single-file components.

## Installation

### Install `vue-language-server` globally

- With npm:

```bash
npm install -g @vue/language-server typescript
```

- With bun (Recommended):

```bash
bun install -g @vue/language-server typescript
```

### Or just add your `vue-language-server` binary to `PATH` if you have it installed locally.

## Command

```text
vue-language-server --stdio
```

## Supported Extensions

`.vue`

## Runtime Requirement

Install `@vue/language-server` and `typescript` so `vue-language-server` is available on `PATH`.
