# marksman-lsp

Marksman language server for Markdown documents.

## Installation

### Install `marksman` globally

- With Homebrew:

```bash
brew install marksman
```

- With Nix:

```bash
nix-env -iA nixpkgs.marksman
```

- With Snap:

```bash
sudo snap install marksman
```

### Or just add your `marksman` binary to `PATH` if you installed it manually.

## Command

```text
marksman server
```

## Supported Extensions

`.md`, `.markdown`

## Runtime Requirement

Install Marksman so the `marksman` binary is available on `PATH`.
